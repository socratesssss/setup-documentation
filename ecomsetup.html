E-Commerce Project Data Modification Guide

Modifying Contact and FAQ Data

This section explains how to modify contact information and FAQ content in your project. You will work with two files:

  • src/data/Contact.tsx - Store contact information
  • src/data/Faqs.ts - Frequently asked questions

Contact Information

1Open Contact.tsx

Navigate to E-Commerce-Project/src/data/Contact.tsx and open it.

2Structure

export const contactInfo = {
  email: {
    icon: <Mail className="w-5 h-5 text-[#0284c7]" />,
    title: "Email",
    lines: ["support@yourstore.com", "sales@yourstore.com"]
  },
  // ... other sections
};

3Update Details

email: {
  lines: ["help@newdomain.com", "orders@newdomain.com"]
},
phone: {
  lines: ["+1 (555) 987-6543", "Mon - Fri: 8am - 7pm"]
},

FAQ Content

1Open Faqs.ts

export const faqs = [
  {
    question: "How long does shipping take?",
    answer: "Most orders ship within 1-2 business days..."
  }
];

2Update or Add FAQs

{
  question: "Do you offer gift wrapping?",
  answer: "Yes, we offer complimentary gift wrapping."
}

Modifying State Data (AddressData.ts)

1Locate AddressData.ts

Find it in E-Commerce-Project/src/data.

2Structure

export const USStates = [
  { name: "Alabama", deliveryCost: 8 },
  { name: "Alaska", deliveryCost: 15 },
];

3Edit Name

{ name: "California", deliveryCost: 12 },
// or add alternative
{ name: "Cali", deliveryCost: 12 }
Changing names may affect existing data. Prefer adding new entries.

4Edit Delivery Cost

{ name: "Texas", deliveryCost: 12 }

5Add New State

{ name: "New Territory", deliveryCost: 14 }

Best Practices

  • Keep data up-to-date
  • Test after every modification
  • Backup files before big changes
  • Use consistent formatting

Troubleshooting

  • Changes not showing: Restart dev server, clear cache
  • Errors: Check commas, brackets, and quotes
  • Icons missing: Verify with Lucide docs
