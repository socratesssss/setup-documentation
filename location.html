Modifying State Data in AddressData.ts

Introduction

This guide explains how to modify state names and delivery costs in the AddressData.ts file located in the E-Commerce-Project/src/data folder.

How to Modify State Data

1Locate the AddressData.ts File

Navigate to the E-Commerce-Project/src/data folder and open the AddressData.ts file in your code editor.

2Understanding the Data Structure

The file contains an array of objects, where each object represents a U.S. state with:

  • name: The state name (string)
  • deliveryCost: The delivery cost in dollars (number)
export const USStates = [
  { name: "Alabama", deliveryCost: 8 },
  { name: "Alaska", deliveryCost: 15 },
  // ... other states
];

3Changing a State Name

To change a state's name, locate the state in the array and modify the name property value: 

// Before
{ name: "California", deliveryCost: 12 },

// After
{ name: "California", deliveryCost: 12 },  // Original remains
{ name: "Cali", deliveryCost: 12 },       // Modified version
Important: Changing state names may affect existing orders or addresses that reference the old name. Consider adding a new entry instead of modifying existing ones.

4Changing Delivery Costs

To update delivery costs, modify the deliveryCost value for the desired state: 

// Before
{ name: "Texas", deliveryCost: 10 },

// After
{ name: "Texas", deliveryCost: 12 },  // Increased from $10 to $12
Tip: Delivery costs should be whole numbers representing dollars.

5Adding a New State

To add a new state, insert a new object to the array following the same format: 

// Add at the end of the array
{ name: "New Territory", deliveryCost: 14 }

6Saving Your Changes

  1. Save the AddressData.ts file
  2. Restart your development server if needed
  3. Test the changes in your application

Best Practices

  • Keep backup of original data before making changes

Troubleshooting

Common Issues

  • Changes not reflecting: Clear cache and restart server
  • TypeScript errors: Ensure all objects follow the correct structure
  • Missing commas: Verify commas between array items
